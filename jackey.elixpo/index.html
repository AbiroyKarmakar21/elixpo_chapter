Commands Under Operation

The discord commands which the users can use for operating the bot and for the usage

/generate

Takes up the prompt and the number of images as the required params and generates images directly

/remix

Takes in the prompt with an image to edit, or upto 3 images to remix the images with a given prompt

/ping & /help

Utilities to show the bot active status and command pallet.
